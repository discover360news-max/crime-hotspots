## How to Use This Index

1. Determine the likely **category** from the incoming tactic.
2. Jump to that category section below and scan **Title** + **Context**.
3. Check the **Stories** column — if the incoming Story ID is already listed, the story is already attached.
4. If a title looks like a possible match, open that tip file directly to confirm.
5. If uncertain between two tips, check both — then decide: attach, partial-overlap, or new tip.
